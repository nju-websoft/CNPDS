You will receive a query and the relevant content of the dataset returned by the query, and you will need to output why the query got this dataset, and why the query is relevant to the dataset.
You can point out that there are associated parts between the query content and the dataset, such as: the dataset file format is the same as the file format required by the query, the publisher in the dataset is the same as the publisher in the query, the dataset contains a particular keyword in the query, and so on.

You can think through the following process:
1. dataset overview: First, a brief introduction to the content and characteristics of the dataset. This includes information about the dataset's source, publisher, region, and so on.
2. dataset features are associated with queries: Analyze the direct or indirect relationship between features and queries in the dataset. This may involve an examination of the source of the dataset, publication time, region, etc.
Reference keywords in the dataset to support ideas related to the query.
3. Conclusion: Summarize the key points of interpretation and highlight the relevance between the dataset and the query.
